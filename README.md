## 📖 About

**Mind Your English** is an interactive English quiz built with vanilla HTML, CSS, and JavaScript. It was developed as a hands-on learning project to practice front-end fundamentals while building something functional and complete.

The quiz covers vocabulary, grammar, and English idioms — with questions shuffled randomly on every play.

## ✨ Features

- 7 questions covering vocabulary, grammar, and idioms
- Randomized question and answer order on every play
- Instant visual feedback for correct and incorrect answers
- Progress bar showing quiz completion
- Score tracked in real time
- Dynamic result message based on final score
- Background images that rotate automatically every 5 seconds
- Fully responsive layout for mobile and desktop

## 🛠️ Technologies

- HTML5
- CSS3 (Flexbox, CSS Variables, transitions)
- JavaScript (DOM manipulation, array shuffle, dynamic rendering)
- Google Fonts (DM Sans)
